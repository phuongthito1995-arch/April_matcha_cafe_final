# April Matcha Café — ITD 210 Capstone Website Project

**Name:** Thi Phuong To  
**Milestone:** Contact Page / Final Milestone before final submission  
**Project Theme:** April Matcha Café, a calm and visual matcha café website.

## What is included

This project includes a complete five-page responsive website:

1. `index.html` — Home page with hero section, feature cards, and daily special interaction.
2. `about.html` — About page with story, values, and FAQ-style accordion.
3. `services.html` — Menu page with interactive tab categories for drinks, desserts, and experiences.
4. `gallery.html` — Gallery page with filter buttons and original visual cards.
5. `contact.html` — Contact page with Formspree placeholder, semantic form, JavaScript validation, inline error messages, success confirmation, FAQ accordion, and live message character counter.

## What is new on the contact page

This milestone adds a responsive and accessible contact page with Formspree integration placeholder, custom JavaScript validation, inline error messages, a live character counter, FAQ accordion, and a success confirmation state.

## Accessibility and interaction

The website includes:

- Semantic HTML structure.
- Mobile-first responsive CSS.
- Visible keyboard focus styles.
- Skip link for keyboard users.
- ARIA attributes including `aria-live` and `aria-describedby`.
- Reduced-motion support in CSS.
- Visible labels for all form fields.
- At least three meaningful interactions across pages, including hover states, mobile menu, back-to-top button, accordions, menu tabs, gallery filters, and contact form validation.
